# 🚀 Cybersecurity Portfolio - Sujampathi Rathnayaka

A modern, minimalist black and white cybersecurity portfolio website with terminal-inspired design and smooth animations.

## ✨ Features

- **Terminal Aesthetic**: Boot screen animation and command-line styling
- **Black & White Design**: Clean, professional minimalist color scheme
- **Smooth Animations**: Scroll-triggered animations and interactive effects
- **Fully Responsive**: Works perfectly on desktop, tablet, and mobile
- **No Dependencies**: Pure HTML, CSS, and JavaScript
- **Performance Optimized**: Lazy loading and efficient animations
- **Dark Mode Ready**: Already in dark theme (black background)

## 🚀 Quick Start

### Option 1: Direct Open
```bash
# Simply open index.html in your browser
start index.html
```

### Option 2: Local Server (Recommended)
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js
npx http-server

# Then visit: http://localhost:8000
```

## 🎨 Customization

### Colors
Edit `src/css/main.css` to change the color scheme:

```css
:root {
    --primary: #000000;
    --secondary: #ffffff;
    --accent: #1a1a1a;
    --text: #ffffff;
    --bg: #000000;
}
```

### Content
Edit `index.html` to update:
- Personal information
- Experience details
- Project descriptions
- Contact information
- Skills and certifications

### Animations
Modify `src/css/animations.css` to adjust animation speeds and styles.

## 📱 Responsive Design
